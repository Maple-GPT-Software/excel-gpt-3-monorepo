# Install ExcelSimplify for Google Sheets

### Step 1: Install ExcelSimplify workspace integration

- Go to Google [workspace marketplace page](https://workspace.google.com/marketplace/app/excelsimplify/357936409669)
- Click on "Install"

### Step 2: Copy start guide to your Google drive

- [Google sheet guide](https://docs.google.com/spreadsheets/d/1OkzknJVDvRdzwObDpzOVZCt2nQwz16PGYvUawvYp__A/edit?usp=sharing)
- This starting point to explore useful features

And that's it! ExcelSimplify can now be used within any of your Google Sheets.
