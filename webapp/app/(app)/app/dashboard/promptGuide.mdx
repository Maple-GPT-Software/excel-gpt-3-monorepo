# Google Sheets Prompt Guide

This prompt guide will help you generate data from Google Sheets using ChatGPT.

## Prerequisites

Before you begin, make sure you have the following:

- A Google Sheets account
- A Google Sheets document with data
- A Google Cloud Platform account
- A Google Cloud Platform project with the Google Sheets API enabled
- A Google Cloud Platform service account key

### Step 1: Enable the Google Sheets API

1. Go to the [Google Cloud Console](https://console.cloud.google.com/).
2. Create a new project or select an existing one.
3. In the left-hand menu, select **APIs & Services** > **Dashboard**.
4. Click on the **Enable APIs and Services** button at the top.
5. Search for **Google Sheets API** and click on it.
6. Click on the **Enable** button.

### Step 2: Create a Service Account Key

1. In the left-hand menu, select **APIs & Services** > **Credentials**.
2. Click on the **Create credentials** button and select **Service account key**.
3. Choose a name for your service account and select **JSON** as the key type.
4. Click on the **Create** button.
5. Your JSON key file will be downloaded to your computer.

### Step 3: Share your Google Sheets document

1. Open your Google Sheets document.
2. Click on the **Share** button in the top-right corner.
3. Add the email address of your service account and give it **Editor** access.

### Step 4: Generate data from Google Sheets using ChatGPT

1. Open your ChatGPT account.
2. Go to the **Google Sheets** section.
3. Select the Google Sheets document you want to generate data from.
4. Enter the Google Cloud Platform service account key that you downloaded in Step 2.
5. Enter the prompt you want to use to generate data.
6. Click on the **Generate** button.

And that's it! ChatGPT will use the Google Sheets API to generate data based on your prompt.
